# Password Generator

A web application that generates secure random passwords based on user preferences. The project provides a simple interface for creating strong passwords that can be used to improve online security.

## Features

* Generate random passwords instantly
* Create multiple passwords at once
* Customize password length
* Includes letters, numbers, and special characters
* Clean and responsive user interface

## Built With

* HTML5
* CSS3
* JavaScript (ES6)
